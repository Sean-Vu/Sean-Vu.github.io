Software Engineering Internship at Inari

As a software engineer intern at Inari, I created new testing framework using Google Test across C++ libraries for cataloging and parsing SQL queries, covering over 1100 lines of code and increasing code coverage by 15%. This testing software helped ensure proper data input from 175 medical devices that tracked blood flow and heart rate in patients with underlying issues. Using this data, I worked with Matplotlib to visualize this data in the form of graphs and diagrams to identify 175 at-risk patients.
